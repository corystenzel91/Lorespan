# Lorespan Mobile GitHub Pages v8.5.2

Complete replacement package for the GitHub Pages publishing folder.

Key changes:
- The first story page stays visible until the reader chooses **Continue to Character Setup**.
- Character setup no longer interrupts the first page automatically.
- Every book uses its own versioned cover asset during opening and return animation.
- The closed flying book is removed before the open spread appears.
- The opening spread and zoom were refined for a smoother transition into the story.
- Fresh books use **Begin Story** instead of **Open Book**.
- New service-worker and asset versions prevent stale animation artwork from being reused.

See `DEPLOY_TO_GITHUB.txt` for replacement instructions and `VALIDATION_v8.5.2.json` for test results.
