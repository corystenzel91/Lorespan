# Lorespan v8.9.28

Upload every file and folder in this package to the root of the GitHub repository. Keep the `movies` folder intact beside `index.html`.

The package contains 20 MP4 movie assets—an opening and return movie for every shelf book. If a browser cannot play a movie, the site automatically uses the procedural 3D fallback animation.

After replacing the repository files, clear the old site cache once or reopen the installed PWA so service worker `lorespan-v8.9.28` takes control.
